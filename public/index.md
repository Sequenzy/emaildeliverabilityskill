# Email Deliverability Skill

emaildeliverabilityskill.com

Email Deliverability Skill for agents that work on email systems.

A compact skill for inbox placement, authentication, reputation, and remediation workflows, built for Codex, Hermes, OpenClaw, Claude Code, Cowork, Goose, Cursor, Windsurf, and other skill-aware agents. It gives agents intake questions, review gates, QA checks, and handoff formats for production email work.

## Install

```bash
npx skills add emaildeliverabilityskill
```

Raw skill folder: `skills/emaildeliverabilityskill`

## Use Cases

- **DNS/authentication audits**: Give agents a bounded way to inspect inputs, propose changes, and surface approval points before anything reaches subscribers.
- **Warmup plans**: Give agents a bounded way to inspect inputs, propose changes, and surface approval points before anything reaches subscribers.
