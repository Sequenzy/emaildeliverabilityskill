# About Email Deliverability Skill

Email Deliverability Skill is a focused skill package and reference site for agents working on inbox placement, authentication, reputation, and remediation workflows. It is designed for practical email operators who need clear intake, QA, approval, and handoff patterns.

The site keeps the public guidance concise while the skill itself gives agents a structured operating routine. The goal is to help agents produce useful email work without skipping safety checks around subscribers, templates, sending systems, and production changes.

Use it with skill-aware tools such as Codex, Hermes, OpenClaw, Claude Code, Cowork, Goose, Cursor, Windsurf, and similar agent environments.
