# Email Deliverability Skill

Static landing page and agent skill for emaildeliverabilityskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add emaildeliverabilityskill
npx skills add Sequenzy/emaildeliverabilityskill --skill emaildeliverabilityskill
```

Raw skill folder: `skills/emaildeliverabilityskill`
